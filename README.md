# GameJam

Unity로 만든 라운드 진행형 2D 액션 게임입니다.

## 게임 설명

플레이어가 적을 공격하면서 제한 시간 동안 버티는 게임입니다.
적에게 계속 맞으면 체력이 줄어들고, 체력이 0이 되면 `BadEnding` 씬으로 이동합니다.

## 조작법

- 이동: 방향키 또는 `A`, `D`
- 공격: `E`

## 주요 기능

- 라운드 씬 3개 구성
- 적 무한 생성
- 적 공격 애니메이션 재생
- 플레이어 체력 200HP
- 적에게 맞을 때마다 10HP 감소
- HP UI 표시
- 체력이 0이 되면 배드엔딩 씬 이동

## 씬

- `Round1`
- `Round2`
- `Round3`
- `BadEnding`

## 개발 환경

- Unity
- C#
